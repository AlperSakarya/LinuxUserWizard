# LinuxUserWizard
A very little tool for conveniently adding/removing Linux users with SSH keys

## Usage:

```curl -L -o linux-user-wizard.sh tinyurl.com/lnxusrw``` <br>
```sudo bash linux-user-wizard.sh```

(adding/removing users requires sudo)
